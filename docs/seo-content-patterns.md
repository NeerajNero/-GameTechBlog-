# SEO Content Patterns

## Article Brief Pattern

Before drafting, define:

- Primary search intent.
- Target reader.
- Main keyword/topic.
- Secondary questions.
- Practical outcome.
- Internal links to include.
- Monetization angle, if any.

Use `templates/article-brief.md`.

## Search Intent Patterns

| Intent | Article Shape |
| --- | --- |
| Buy | Buying guide, comparison table, recommendation by budget/use case |
| Fix | Troubleshooting guide with symptoms, causes, fixes, prevention |
| Learn | Explainer with examples and clear takeaways |
| Compare | Side-by-side comparison and final recommendation |
| Setup | Step-by-step guide with checklist |

## Heading Pattern

- H1: article title from page.
- H2: main reader questions.
- H3: steps, options, or sub-decisions.
- Avoid decorative headings that do not help scanning.

## Internal Linking Pattern

- Link from broad guides to specific guides.
- Link from specific guides back to broader category pages.
- Add related articles by category and tags.
- Update older articles when publishing a new related guide.

## Refresh Pattern

Prioritize updates for:

- Buying guides.
- Product recommendations.
- Platform setup instructions.
- Articles affected by new console, GPU, laptop, or storage releases.
