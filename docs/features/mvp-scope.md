# MVP Scope

## In Scope

- Next.js App Router web app.
- TypeScript.
- Tailwind CSS.
- MDX article rendering.
- Local article files in `content/articles/`.
- Frontmatter metadata.
- Homepage.
- Article index.
- Article detail pages.
- Category pages.
- Tag pages.
- About/contact/policy/disclosure pages.
- SEO metadata.
- Sitemap and robots.
- Article structured data.
- Social preview metadata.
- Vercel deployment readiness.

## Out Of Scope

- Backend.
- Database.
- Prisma.
- NestJS.
- Docker.
- Auth.
- Admin panel.
- CMS.
- Payments.
- User accounts.
- Comments.
- Runtime article editor.
- Price tracking.
- Product database.

## First Build Target

The first implementation task should produce a working local blog shell with at least one sample MDX article and the content helpers needed to list and render articles.

## Done Criteria

- Site runs locally.
- Published MDX articles render.
- Draft MDX articles are excluded.
- Homepage, article listing, article detail, category, and tag routes work.
- SEO metadata exists for key pages.
- Build succeeds.
